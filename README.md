# Pet Lovers Community Onboarding Website

This is a full-stack web application for a pet lovers community group. It includes an onboarding wizard for new users, user profiles, and the ability to view other users' profiles.

## Features

*   **Onboarding Wizard:** A 5-step wizard to guide new users through setting up their profile.
*   **User Profiles:** Each user has a profile with their personal and pet information.
*   **View Profiles:** Users can view the profiles of other members in the community.
*   **Edit Profile:** Users can edit their profile information, including their password and photos.
*   **Delete Profile:** Users can delete their profile from the community.
*   **Login/Logout:** Standard user authentication.

## Technologies Used

*   **Backend:** PHP
*   **Frontend:** HTML, CSS, JavaScript, Bootstrap
*   **Database:** CSV flat-file

## How to Run

1.  **Web Server:** You need a web server with PHP support (e.g., Apache, Nginx).
2.  **Clone the Repository:** Clone this repository to your web server's document root.
3.  **Permissions:** Ensure that the `uploads` directory is writable by the web server.
4.  **Start:** Open your web browser and navigate to the project's URL.

### Using PHP's Built-in Web Server

For development purposes, you can use PHP's built-in web server.

1.  Open your terminal or command prompt.
2.  Navigate to the project's root directory.
3.  Run the following command:

    ```bash
    php -S localhost:8000
    ```

4.  Open your web browser and go to `http://localhost:8000`.

Now you can test the application.
